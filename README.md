# Kampala Youth Skill Matcher + Dashboard

🚀 **Live Demo:** [uganda-youth-skill-matcher.streamlit.app](https://uganda-youth-skill-matcher-7dmqbavjnexa7kg27jqm9.streamlit.app)

Interactive dashboard analyzing employment, digital skills, and vocational training for 150 youth across Kampala's 5 divisions: Central, Kawempe, Makindye, Nakawa, and Rubaga.

---

## Features
- Filter by division, gender, employment status
- Unemployment rate & digital skills analytics  
- Skills-to-jobs matching recommendations
- Interactive map of Kampala divisions

## Tech Stack
Python · Streamlit · Pandas · Plotly · Folium
